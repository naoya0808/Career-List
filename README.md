# CareerList（キャリアリスト）

## サービス概要

CareerListは、就職活動や転職活動の際に、自分の受けている企業の情報をまとめて、リスト化することができるアプリケーションです。<br>
就職活動や転職活動を行う上で、企業情報のまとめ方やまとめる時間を独自で考えることは手間がかかると思います。そんな悩みを自身だけでなく他者も抱えてると感じたので作成しました。<br>
すべての就職活動・転職活動を行う方々をCareerListは応援します。

## 主要機能

### ユーザー機能

・ユーザー登録と削除<br>
・メールアドレスとパスワードの変更

### 投稿機能
・企業の新規作成と投稿<br>
・企業の志望度を表す星評価<br>
・投稿の閲覧と個別表示<br>
・投稿の編集と削除<br>
・投稿削除の確認ダイアログ<br>
・ページネーション<br>
・ダミーデータ<br>
